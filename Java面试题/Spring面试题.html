    # Spring其实没什么好问的

    1、因为Spring只是一个Bean工厂，没了。

    2、还能问点什么？也就只有循环依赖了，其实就是问依赖注入，没了。

    3、还能问点什么？也就只有AOP了，AOP其实就是动态代理，没了。

    # Spring自动注入

    1、自动注入（又称自动装配）是依赖注入的一种形式。

    2、Spring自动注入相关源码：

    org.springframework.beans.factory.annotation.AutowiredAnnotationBeanPostProcessor#buildAutowiringMetadata

    org.springframework.beans.factory.annotation.AutowiredAnnotationBeanPostProcessor#postProcessProperties

    org.springframework.beans.factory.annotation.InjectionMetadata.InjectedElement#inject

    3、可知，自动注入的逻辑为：先通过反射收集@Autowired注解，再通过反射赋值。

    # Spring之IOC

    1、控制反转inversion of control，是一种编程思想（又称设计模式）。

    2、实际上很多设计模式都用到了控制转移的思想。但在Java面试界，Spring的IOC特指依赖注入。
