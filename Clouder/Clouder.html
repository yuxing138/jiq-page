# 项目介绍

1、开源地址：https://github.com/yuxing138/clouder.git
2、开源目的：方便自己将本地文档云化。

# 使用步骤
# 1、新建配置文件

windows：windows.properties
macos：macos.properties

# 2、配置参数
#根目录
pageRootDirectory=/Users/mac/Desktop/temp
#图片库
pictureRepository=https://github.com/yuxing138/picture.git
pictureDomain=https://picture.yuxing138.top
#文档库
documentRepository=https://github.com/yuxing138/document.git
pictureDomain=https://document.yuxing138.top
#令牌密钥、日志级别
privateTokenPassword2=加密密码2
printLevel=info

# 3、生成密钥

通过java -jar DocumentClouder.jar -t生成密钥
privateToken为：github仓库的token
privateTokenPassword1为：token加密密码1
privateTokenPassword2为：token加密密码2

# 4、上传或更新文档

上传文档：拷贝文档到pageRootDirectory/upload/topic/xx.md，执行命令java -jar DocumentClouder.jar即可。
更新文档：拷贝文档库中的文档到pageRootDirectory/upload/topic/xx#xxxx.md，执行命令java -jar DocumentClouder.jar即可。
镜像文档：拷贝文档到pageRootDirectory/upload/topic/中，执行java -jar DocumentClouder.jar -s即可。（注意：“原始库没有，文档库有”的文档将被删除，即文档库是原始库的镜像。)
